## 🧑‍💻 About Me

> *"I don't just build features — I architect solutions that scale gracefully under pressure."*

I'm a **Senior Software Engineer** at **[Brain Station 23](https://brainstation-23.com)**, Dhaka, Bangladesh, with **7+ years** of professional experience building production-grade systems that power **millions of daily users**. I specialize in distributed backend architecture, microservices, and full-stack development — taking projects from concept to massive scale.

- 🏢 **Now:** Senior SWE @ [Brain Station 23](https://brainstation-23.com) *(Mar 2025 – Present)* — Leading backend for **[MyGP App](https://play.google.com/store/apps/details?id=com.portonics.mygp)** with **23M+ registered users**, built on Laravel, Lumen, Node.js & Kubernetes
- 💼 **Previously:** Software Engineer @ [weDevs](https://wedevs.com) *(Dec 2021 – Feb 2025)* — Architected **[Appsero](https://appsero.com)** (serving 10K+ WordPress products) & **[weMail](https://getwemail.io)** (millions of emails delivered)
- 🏗️ **Before that:** Web Application Developer @ [Alesha Tech](https://aleshamart.com) *(Apr 2021 – Nov 2021)* — Optimized high-traffic e-commerce & LMS platforms
- 📦 **Open Source:** Author of [laravel-self-updater](https://github.com/anisAronno/laravel-self-updater), [multipurpose-admin-panel-boilerplate](https://github.com/anisAronno/multipurpose-admin-panel-boilerplate), [laravel-starter](https://github.com/anisAronno/laravel-starter)
- 🌱 **Currently exploring:** Go · GraphQL · Advanced Kafka streams · Kubernetes operators
- 🏆 **Highlights:** Mentored 20+ developers · 1,900+ contributions/year · 153+ public repositories · Reduced release cycles by 60% with modern CI/CD pipelines
- 📄 **[Download CV](https://github.com/anisAronno/anisAronno/raw/main/anisAronno.pdf)**
